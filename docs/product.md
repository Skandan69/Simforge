# Product notes

## Current status

SimForge is at the foundation stage. The repository establishes the delivery shape without committing to product behavior.

## Working intent

SimForge will provide a place to build, run, and understand simulations. The problem definition, target user, and first workflow remain to be decided before feature development begins.

## Before product implementation

Document and agree on:

1. Primary user and their current workflow.
2. First simulation domain and its boundaries.
3. Inputs, outputs, and success criteria.
4. Collaboration and persistence requirements.
5. The smallest end-to-end release worth testing.
